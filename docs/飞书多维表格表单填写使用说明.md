# 飞书多维表格表单填写使用说明

这份说明给上传图片素材的同事使用。只需要按表单填写，不需要了解系统怎么处理图片。

## 一句话说明

把图片上传到飞书表单，并填写清楚项目、图片名字、描述、来源和是否可商用。  
系统会自动把符合条件的图片整理进 PPT 素材库。

## 需要填写的 7 项

| 字段 | 怎么填 | 示例 |
| --- | --- | --- |
| 项目 | 填图片所属项目。没有明确项目可留空。 | `minduck`、`董办`、`均胜智能座舱` |
| 名字 | 填图片的用途或内容名称，不要再写成 `项目-名字`。 | `产品图标`、`领导生活照` |
| 描述 | 说明图片是什么、适合什么场景、时间或特殊备注。 | `2026 年 minduck 项目图标`、`董办活动现场照片` |
| 文件 | 上传图片文件。 | `logo.png`、`活动照片.jpg` |
| 来源 | 选择图片来源。 | `下载`、`拍摄`、`截图`、`AI生成`、`渲染` |
| 来源二级分类 | 暂定选择或填写 `计划`。 | `计划` |
| 是否可商用 | 选择这张图片是否已经确认可用于公司 PPT 或对外材料。 | `是`、`否` |

## 项目怎么填

`项目` 决定图片最后进入哪个素材文件夹。

### 1. 有明确项目就填写项目

示例：

```text
董办
minduck
智能制造
```

这样填写后，系统会把图片放入对应项目的素材文件夹。  
如果这个项目文件夹还不存在，系统会自动创建。

### 2. 没有明确项目可以留空

这样填写后，系统会把图片放入通用素材库。

## 名字怎么填

`名字` 只写图片本身叫什么，不再包含项目名前缀。

推荐：
```text
会议室背景
通用科技感底图
流程箭头图标
产品图标
领导生活照
```

不推荐：

```text
董办-领导生活照
minduck-产品图标
```

原来的 `项目-名字` 写法已经废弃，请把项目信息填到 `项目` 字段。

## 描述怎么写

描述尽量写给以后做 PPT 的人看。

推荐写：

```text
2026 年 minduck 项目图标，适合用于项目介绍页、封面页。
董办活动现场照片，适合用于企业文化、活动回顾页面。
浅色科技感背景图，适合用于产品方案、架构说明页面。
```

不推荐只写：

```text
图片
素材
好看的图
随便用
```

描述越清楚，后续 AI 识别和 PPT 调用越准确。

## 是否可商用怎么选

这个字段很重要。

| 选项 | 含义 | 系统会怎么处理 |
| --- | --- | --- |
| 是 | 已确认可以用于公司 PPT、汇报材料或对外材料 | 会进入图片处理和素材库 |
| 否 | 不确定版权、不能商用、仅供参考或暂时不想入库 | 会跳过，不进入素材库 |

注意：系统不会替你判断版权。  
如果你不确定，就选 `否`。

## 文件上传要求

请上传常见图片格式：

```text
png / jpg / jpeg / webp
```

建议：

- 一条记录上传一张主要图片。
- 图片尽量清晰，不要上传太小、太糊、带大量无关边框的截图。
- 如果一组图片属于同一个项目，统一填写相同的 `项目` 字段。

## 常见问题

### 我填完以后图片会马上出现吗？

不一定。系统需要等管理员或机器人启动处理流程后，才会统一拉取和整理。

### 我填错了怎么办？

在系统处理前，直接修改表单对应记录即可。  
如果已经处理完成，联系管理员重新处理或手动调整。

### 为什么我上传的图片没有进入素材库？

优先检查这几项：

- `文件` 是否真的上传了图片。
- `是否可商用` 是否选择了 `是`。
- `名字` 是否填写了。
- 图片格式是否是常见图片格式。

### 来源选什么？

- 自己拍的照片：选 `拍摄`
- 网上下载的图片：选 `下载`
- 从屏幕截取的图片：选 `截图`
- AI 生成的图片：选 `AI生成`
- 3D 渲染、效果图、设计稿渲染图：选 `渲染`

### 来源二级分类怎么填？

目前先统一填写或选择：

```text
计划
```

后续如果来源体系细化，再按新的选项填写。

## 最后检查

提交前确认：

- 名字写清楚了。
- 项目填在 `项目` 字段里，没有写进 `名字`。
- 描述能让别人看懂图片用途。
- 文件已经上传。
- 来源选择正确。
- 来源二级分类已经填写。
- 是否可商用已经选择 `是` 或 `否`。
